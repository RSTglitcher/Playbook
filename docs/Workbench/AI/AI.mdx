AI in the Workbench splits into **two complementary groups**:

### **1. AI Assistants**
Cognitive partners you work *with*:

- Microsoft Copilot
- Claude
- ChatGPT
- GitHub Copilot
- Augment

These are **thinking environments** — tools that amplify reasoning, writing, coding, and analysis.

### **2. AI Skills**
Composable **intelligence infrastructure** used inside AI systems:

- Embeddings
- Vector Stores
- Tokenisation
- Model Serving
- Evaluation

These behave like frameworks:
you combine them to build pipelines, retrieval systems, and intelligent features.

### How to use this directory
Use these pages to understand:

- how to think with AI assistants
- how AI skills combine into pipelines
- how modern AI systems are constructed
- how to reason about retrieval, inference, and drift
- how to diagnose issues across the AI stack

AI defines **how intelligence is built and how you collaborate with it**.

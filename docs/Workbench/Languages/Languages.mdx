Languages are **distinct cognitive environments** — each one shapes how you think, design, and express logic.
You typically work in **one language at a time**, because each has its own:

- syntax
- idioms
- mental models
- runtime behaviour
- ecosystem

Languages are **exclusive modes of thought**.
Switching languages means switching how you reason.

### What belongs here
- Java
- Python
- TypeScript
- Scala
- SQL

### How to use this directory
Use these pages to understand:

- how to think in each language
- its idioms and patterns
- its strengths and constraints
- how engineers use it
- how analysts interpret it

Languages define **how you express logic**.

## Open

```shell
open ~/.zshrc
open ~/.bash
```

**Shortcuts**

| Keyboard Shortcut | Purpose                 |
|-------------------|-------------------------|
| Ctrl + K          | Clear screen            |
| Ctrl + W          | selete 1 word at a time |


## Navigation

| Bash command | Purpose                                    |
|--------------|--------------------------------------------|
| pwd          | print working directory contents           |
| ls           | list contents                              |
| ls -l        | list detailed                              |
| ls -a        | list hidden files too                      |
| ls -r        | list in reverse order                      |
| ls -R        | recursive list, searches lower directories |
| ls -i        | lists the inside                           |
| tree         | show file directory tree                   |
| cd           | change directory                           |
| cd ..        | move up one                                |
| man command  | manual page                                |
| history      | previous commands                          |
| clear        | clear screen                               |
| ctrl + C     | interrupt current                          |
| ctrl + D     | logout                                     |
| ctrl + Z     | suspend process                            |

## Process Management

| Bash command    | Purpose                             |
|-----------------|-------------------------------------|
| ps              | list processes                      |
| ps -f           | Full list of all processes          |
| top             | real-time & process activity        |
| kill_process_id | terminate                           |
| jobs -l         | List of background and stopped jobs |
| bg              | put in background                   |
| fg              | put in foreground                   |
| Ctrl C          | Kill                                |
| Ctrl Z          | Pause                               |
| fg % jobid      | Continue in Foreground              |
| bg % jobid      | Continue in Background              |

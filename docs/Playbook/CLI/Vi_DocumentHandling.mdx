#Vi 

## Universal UNIX editor
- Command mode: Keystrokes are interpreted as an instruction rather than a letter to be inserted.
- Input mode: Character typed by user is inserted into the document.

| Command       | Description                 |
|---------------|-----------------------------|
| vi filename	  | Creates/Opens file          |
| vi R filename | Opens read only             |
| view filename | Opens existing in read only |

## Vi Editor
When you enter VI, you start in command mode.  Pressing esc will always bring you back to command mode.

| Command | Description                                                     |
|---------|-----------------------------------------------------------------|
| i	      | enter insert mode and insert before the current character       |
| I	      | enter insert mode and insert at the end of the current line     |
| @	      | input mode                                                      |
| a       | append (insert after current character)                         |
| A	      | append at the end of the current line                           |
| o	      | text mode - new line below cursor                               |
| O	      | text mode  - new line above cursor                              |
| :#	     | move to line #.  Example: 8 will move you to line 8             |
| D	      | deletes the text from the cursor to the end of the current line |
| dd	     | delete the current line                                         |
| dw	     | deletes current word                                            |
| gg	     | move to top of file                                             |
| 4gg     | 	move to line 4                                                 |
| G 	     | move to end of file                                             |
| r	      | replace one character (and return to command mode)              |
| R	      | enter replace mode                                              |
| u	      | undo last change                                                |
| U	      | undo all changes to the current line                            |
| x	      | deletes the current character                                   |
| ZZ	     | save your work and exit                                         |
| /word   | 	highlight a particular word throughout your file               |


## Command Mode - ESC - edits text written
| Command         | Description                            |
|-----------------|----------------------------------------|
| :w	             | write out data from buffer to file     |
| :wq	            | save your work and exit                |
| :x              | write out data and exit                |
| :q              | quit editor as long as no changes made |
| :q!             | exit vi without saving                 |
| x	              | delete ch                              |
| nx	             | delete ch if n                         |
| dw	             | delete word                            |
| ndx	            | delete word if n                       |
| dd	             | delete sentence                        |
| ndd	            | delete sentence if n                   |
| :1	             | first line                             |
| :n	             | nth line                               |
| G	              | bottom                                 |
| ^	              | beginning of current line              |
| $	              | end of current line                    |
| r	              | replace single character               |
| R	              | overtype mode                          |
| .-	             | repeat previous command                |
| /searchString	  | search for                             |
| ?searchString	  | reverse search                         |

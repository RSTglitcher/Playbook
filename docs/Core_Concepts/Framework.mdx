A programming framework is a set of tools to make it easier to perform common programming tasks.

	- Framework > Libraries > Code

" A pre-packaged set of solutions that solves common development problems"

	1. Front end frameworks: Common problems = Routing & Templating. Therefore developers use web frameworks for these.
	- MVC framework (Model-View-Controller) = Separating the 3 key concerns into well defined layers.
	- React Framework = HTML, CSS + JavaScript.

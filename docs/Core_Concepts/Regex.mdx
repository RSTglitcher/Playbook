## [Regular Expression](https://regex101.com/)

Define patterns to search, match and manipulate text strings.

## Use Cases

- Text search & filtering
- Data Validation
- Data Extraction
- Text Replacement & Formatting
- Log Analysis
- URL Routing & Rewriting
- Parsing
- Refactoring
- Security
- Automated text filtering
- Validations
- Cleaning
- Data Masking

---

## Examples

- **Social Security:** ^\d{3}-\d{2}-\d{4}$
- **Email Address:** \b\wt@\wt\.\w+\b
- **Email:** ^[a-zA-Z0-9\.\_]*[a-zA-Z0-9]+[\.\_]*\@[a-zA-Z0-9\.\_]*[a-zA-Z0-9]+[\.\_]*\.[a-zA-z]+
- **Colours:** (red|blue|green)
- **IPv4 Address:** \d{1,2}\.\d{1,3}\.\d{1,3}\\d{1,3}

## Characters

| Regex | Purpose                            |
|-------|------------------------------------|
| .     | single character not newline       |
| \d    | digit (0-9)                        |
| \w    | word                               |
| \s    | whitespace                         |
| \S    | non-whitespace                     |
| \     | Escape special to treat as literal |

---

## Quantify patterns

| Regex     | Purpose                                             |
|-----------|-----------------------------------------------------|
| (n)       | # occurrences                                       |
| (n,empty) | at least # occurrences                              |
| (n,m)     | between # occurrences                               |
| .+        | At least 1 character                                |
| +         | Repeats a character one or more times (non-greedy)  |
| ++        | Repeats a character one or more times (greedy)      |
| +?        | Repeats a character one or more times               |
| ?         | zero or one occurrences                             |
| .*        | Between 0 and more characters                       |
| *         | Repeats a character zero or more times (non-greedy) |
| **        | Repeats a character zero or more times (greedy)     |
| *?        | Repeats a character zero or more times              |
| ?         | Once or None                                        |
| ^         | Matches start of the line                           |
| $         | Matches end of the line                             |
| .         | Matches any single character                        |
| [^]       | Single character with any not in range              |

## Specific Patterns

| Regex      | Purpose                                                            |
|------------|--------------------------------------------------------------------|
| (abc)      | exact sequence                                                     |
| (a pipe b) | a or b                                                             |
| (^abc)     | Except a, b, or c                                                  |
| (a-z)      | Lowercase                                                          |
| (A-Z)      | Uppercase                                                          |
| (?i)       | Case-insensitive mode                                              |
| (?s)       | Single-line mode                                                   |
| (?m)       | Multi-line mode                                                    |
| (?x)       | Free-spacing mode - comment/whitespace mode                        |
| [aeiou]    | Matches a single character in the listed set                       |
| [^XYZ]     | Matches a single character not in the listed set                   |
| [a-z0-9]   | The set of characters can include a range                          |
| (          | Indicates where string extraction is to start                      |
| )          | Indicates where string extraction is to end                        |
| M          | Multiline                                                          |
| ()         | Capturing group                                                    |
| []         | Declare a regular expression with character ranges inside brackets |
| [^]        | Negates characters in brackets e.g. [^3] means not 3.              |
| (3)        | Exactly 3 times - squiggly brackets, not parenthesis               |
| (2,4)      | Two to 4 times - squiggly brackets, not parenthesis                |
| (3, )      | Three or more times - squiggly brackets, not parenthesis           |
| (0-9)      | Digit                                                              |

## Modifiers

| Regex | Purpose                        |
|-------|--------------------------------|
| i     | Case insensitive               |
| g     | Global matching                |
| m     | Multi-line matching            |
| -w    | Match whole word               |
| -v    | Returns everything but matcher |
| -i    | Ignore Case                    |
| -n    | Line Number                    |
| -c    | Count Matching Lines           |
| -B    | Before                         |
| -A    | After                          |
| -C    | Context (before + after)       |

## Grep REGEX (basic = grep) [-options]
| Command | Description                              |
|---------|------------------------------------------|
| -w      | Match the whole word                     |
| -v      | Returns everything but the matched value |
| -i      | Ignore case                              |
| -n      | Returns line number                      |
| -c      | Count matching lines                     |
| -B      | Before                                   |
| -A      | After                                    |
| -C      | Context (before & after)                 |

## eGrep REGEX (extended = egrep) [-options]

| Command     | Description                                     |
|-------------|-------------------------------------------------|
| !           | Negate test i.e. Not true                       |
| 2 pipes     | OR                                              |
| -o          | Shell syntax for OR                             |
| &&          | AND                                             |
| -a          | Shell syntax for AND                            |
| *           | Match 0 or more                                 |
| ?           | Match 1 character                               |
| []          | Anything in range                               |
| {}          | Lists                                           |
| [a-e]       | Match a-e                                       |
| [!0-9]      | Anything except 0-9                             |
| ^           | Match at start of line                          |
| $           | Match at end of line                            |
| []          | Match anything in [] position 1 only            |
| .           | Single character                                |
| [^]         | Single character with any not in range          |
| *           | Zero/more occurrences of preceeding character   |
| .*          | Any number of characters                        |
| \           | Escape meta character, treat as literal         |
| +           | One or more occurrences of preceding character  |
| ?           | Match 0 or 1 occurrences of preceding character |
| (a pipe b ) | Match a or b                                    |

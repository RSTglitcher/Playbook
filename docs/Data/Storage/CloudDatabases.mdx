<Tabs>
<TabItem value="bigQuery" label="BigQuery">

As part of the development of BigQuery, the team designed an abstraction (MapReduce) that allows complex computations to be expressed in terms of two steps:
- `map function` - Processed a key/value pair to generate a set of intermediate key/value pairs
- `reduce function` - Merged all intermediate values associated with the same intermediate key

This led to the development of **Apache Hadoop**.
